Dự án cá nhân

Web Phòng Khám Online

Mô tả: Hệ thống web giúp bệnh nhân đặt lịch hẹn với bác sĩ, quản lý hồ sơ bệnh án và hỗ trợ tư vấn trực tuyến.

Chức năng chính:

  • Đặt lịch hẹn: Chọn bác sĩ, thời gian khám và đặt lịch trực tuyến.
  • Quản lý hồ sơ: Lưu trữ thông tin bệnh án, đơn thuốc, lịch sử khám chữa bệnh.
  • Hệ thống đăng nhập: Phân quyền cho bệnh nhân, bác sĩ, quản trị viên.
  • Tư vấn trực tuyến: Chat trực tiếp với bác sĩ hoặc đặt câu hỏi.
  • Thanh toán online: Hỗ trợ thanh toán qua thẻ ngân hàng, ví điện tử.

Công nghệ sử dụng: ASP.NET Core, Entity Framework, SQL Server, HTML, CSS, JavaScript, Bootstrap.
